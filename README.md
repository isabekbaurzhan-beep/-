Бұл жоба — JavaScript, HTML және CSS көмегімен жасалған қарапайым іздеу жүйесі. Қолданушы іздеу жолағына мәтін енгізген кезде, тізім автоматты түрде сүзгіден өтіп, сәйкес нәтижелер бірден көрсетіледі (live search). Жоба интерфейсі жеңіл және түсінікті, бастапқы деңгейдегі веб-дамыту дағдыларын көрсетуге арналған.
